
  <h1>🌠 Curseur Project</h1>

  <p><strong>Curseur Project</strong> is a creative and interactive web experience featuring three distinct animated cursor styles:</p>

  <ul>
    <li><strong>💫 Curseur Comet</strong></li>
    <li><strong>🌸 Curseur Euphoria</strong></li>
    <li><strong>🔥 Curseur Ragnarök</strong></li>
  </ul>

  <p>Each version features animated cursor trails, visual feedback on click, and themed backgrounds for an engaging user experience.</p>

  <h2>✨ Features</h2>

  <h3>💫 Curseur Comet</h3>
  <ul>
    <li>Animated space background with stars</li>
    <li>3D cube animation</li>
    <li>Custom comet trail following the mouse</li>
    <li>Interactive click effect</li>
  </ul>

  <h3>🌸 Curseur Euphoria</h3>
  <ul>
    <li>Soft glowing cursor</li>
    <li>Shooting star background</li>
    <li>Expanding animation on click</li>
  </ul>

  <h3>🔥 Curseur Ragnarök</h3>
  <ul>
    <li>Fiery trail effects with intense energy</li>
    <li>Dark UI theme</li>
    <li>Fireball bursts and randomized particles</li>
  </ul>

  <h2>🛠 Technologies Used</h2>
  <ul>
    <li>HTML5, CSS3</li>
    <li>JavaScript, jQuery</li>
    <li>DOM Manipulation</li>
    <li>Custom cursor + animation events</li>
  </ul>
